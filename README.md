# To-Do App 📝

This project is a basic to-do list mobile application built with the **Flutter framework** and **Dart**. It allows users to add, manage, and archive tasks. This app was made for learning purpose so it lacks some features such as storing memory like a normal To-Do list app.

## ✨ Features

-   **Add Tasks:** Easily add new tasks with a title, due date, and priority level.
-   **Manage Tasks:** Tasks are displayed in a clean, scrollable list on the home screen.
-   **Archive Completed Tasks:** Completed tasks are moved to a separate archive view after a short delay.
-   **Priority Display:** Tasks are visually color-coded based on their priority (High, Medium, Low).
-   **Navigation:** A simple bottom navigation bar allows you to switch between the main task list and the archive.
